Lista Completa de Melhorias e Correções

Correções de Bugs Críticos
    - Sistema de XP com Habilidades
    - Corrigido o bug onde o XP não era concedido ao derrotar inimigos usando habilidades especiais
    - Implementado retorno de inimigos derrotados pela habilidade
    - Clone Corrompido
    - Corrigido dano incorreto (agora usa atributos equivalentes ao jogador)
    - Implementado uso de habilidades pelo clone
    - Adicionado cálculo de XP adequado
    - Ação Inválida em Combate
    - Corrigida a lógica onde inimigos não atacavam após ações inválidas
    - Implementado sistema de "turno perdido" como punição

Melhorias de Gameplay
    - Sistema de Habilidades
    - Limitado uso a 1 vez por combate
    - Efeitos visuais melhorados para cada classe:
    - Mago: Dano em área
    - Paladino: Cura significativa
    - Arqueiro: Alto dano único
    - Guerreiro: Ataque devastador
    - Balanceamento
    - Aumentado XP do boss final (Velkar)
    - Rebalanceamento de atributos de inimigos
    - Progressão de nível mais satisfatória
    - Sistema de Combate
    - Inimigos agora atacam imediatamente após:
    - Uso falho de habilidade
    - Ação inválida
    - Erro de input

Melhorias Visuais e UX
    - Layout do Personagem

    - Bordas estilizadas
    - Alinhamento profissional
    - Seções organizadas
    - Fluxo de Jogo
    - Telas de transição melhoradas
    - Timing de pausas ajustado
    - Mensagens mais imersivas
    - Feedback Visual
    - Efeitos de dano mais claros
    - Indicadores de turno
    - Destaque para eventos importantes

Melhorias Técnicas
    - Organização de Código

    - Separada lógica de combate
    - Modularização melhorada

    - Vida +10 por nível

    - Inputs inválidos tratados corretamente
    - Prevenção de crashes em combates
